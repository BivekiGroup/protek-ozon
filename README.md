## Ozon API Demo

This project includes a simple demo page that calls Ozon Seller API `POST /v3/product/list` and renders the raw JSON response.

Setup:

- Copy `.env.local.example` to `.env.local` and set your credentials:

```
OZON_CLIENT_ID=your_client_id_here
OZON_API_KEY=your_api_key_here
```

- Start the dev server:

```
npm run dev
```

- Open `http://localhost:3000/ozon/products` to fetch and view the response.

Internally, the app hits `src/app/api/ozon/products/route.ts`, which proxies your request to Ozon with the required headers and returns the upstream JSON.
